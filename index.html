Commands

The following table includes a list of all available commands to Ekko users, along with some extra information.

Command Aliases Arguments Description
play p url or title Add an item to the guild queue.
start N/A N/A Start the current guild queue if the bot is not currently paused.
stop leave N/A Stop all audio playback from Ekko
This will clear the guild queue.
skip N/A N/A Skip the song that is currently playing.
pause N/A N/A Pause the audio playback within the guild.
resume N/A N/A Resume the audio playback within the guild.
move summon, join N/A Move Ekko from one voice channel to your current voice channel.
queue q, playing, np, nowplaying N/A View all items that are currently in the quild queue.
Queue control
remove N/A queue id or all Remove a queue item from the current guild.
loop N/A N/A Toggle looped audio playback within the current guild.
shuffle N/A N/A Toggle the shuffle functionality on the queue within the current guild.
continous autoplay N/A Toggle continuous video playback, automatically queue the first recommended YouTube video.
Playlist control
list playlists, library N/A List all the playlists saved to your account.
view N/A playlist id or name View the contents of a specific playlist.
save keep, export playlist name Save the current guild queue as a playlist.
load import playlist id or name View the contents of a specific playlist.
delete N/A playlist id or name Delete one of your saved playlists.
